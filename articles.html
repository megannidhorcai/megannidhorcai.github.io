Murder in the USA - Thesis (77%)

Maths University

The Murder Accountability Project (MAP) is a non-profit group established in 2015 to track America’s unsolved homicides. The organisation tasks itself with educating the American public on the value of accurate data collection relating to unsolved homicides within the United States (US). The project focuses on 2 main objectives: • What factors relating to the victim influence the success or failure of a murder investigation; can a predictive model be created for this? This objective focuses on building and improving different predictive classifiers; Naive Bayes, Neural Networks, k- Nearest Neighbours, with a particular emphasis on Logistic Regression using model selection methods within regression based models. • Is it possible to find a serial killer in the data using cluster analysis? The second objective focuses on creating a dissimilarity matrix between a sample of cases, to further interrogate and find a cluster using both agglomerative and divisive methods for identification.
